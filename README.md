# JSON_data_editer

　キーを設定し、それに対応する値を追加していく形でJSONを作成するためのHTMLファイルです。既存のJSONファイルをインポートまたは、自分で1からキー設定と値を入力していくことでJSONファイルを作成できます。  
 作成したデータを保存するためには、JSONとしてエクスポートする必要があります。  
 LLMの学習データとなるJSONファイルを自前で作れるようにするために作成いたしました。GitHub Pagesで使えるように、Claude Codeに作ってもらったHTMLファイルのみをリポジトリに置いています。
